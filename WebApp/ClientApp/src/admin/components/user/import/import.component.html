Import Users

Usage: Each line contains four parts separated by commas, represents a new user to import.

  1. Contestant ID (string)
  2. Contestant Name (string)
  3. Email (email)
  4. Password (string, optional)

Examples:

  • 171860001,ZhangSan,171860001@smail.nju.edu.cn create a user with contestant ID as password.
  • 171860001,ZhangSan,171860001@smail.nju.edu.cn,password create a user with specified password.
