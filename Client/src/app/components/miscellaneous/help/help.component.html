编程语言

  • C 11 [1.0x]

    C 11 [1.0倍运行时间限制]

    • 编译环境：GCC 9
    • 编译标准：GNU C 11
    • 编译参数：-std=c11 -static -march=native -O2 -fno-strict-aliasing -DONLINE_JUDGE
    • 参考程序： 
      #include <stdio.h>
int main() {
  int a, b;
  scanf("%d%d", &a, &b);
  printf("%d\n", a + b);
  return 0;
}
  • C++ 17 [1.0x]

    C++ 17 [1.0倍运行时间限制]

    • 编译环境：GCC 9
    • 编译标准：GNU C++ 17
    • 编译参数：-std=c++17 -static -march=native -O2 -fno-strict-aliasing -DONLINE_JUDGE
    • 参考程序： 
      #include <iostream>
int main() {
  int a, b;
  std::cin >> a >> b;
  std::cout << a + b << std::endl;
  return 0;
}
  • Java 11 [2.0x]

    Java 11 [2.0倍运行时间限制]

    • 编译环境：AdoptOpenJDK 11
    • 编译要求：类名必须为Main
    • 编译参数：-J-Xms64m -J-Xmx512m
    • 参考程序： 
      import java.io.*;
import java.util.*;
    
public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        PrintWriter out = new PrintWriter(System.out);
    
        int a = in.nextInt();
        int b = in.nextInt();
        out.println(a + b);
        out.flush();
    }
}
  • Python 3 [5.0x]

    Python 3 [5.0倍运行时间限制]

    • 运行环境：Python 3.8+
    • 注意事项：Python代码在执行前会使用Pylint进行检测并编译为pyc文件，如果Pylint认为代码不正确则视为编译错误
    • 参考程序： 
      print(sum(int(x) for x in input().split(' ')))
