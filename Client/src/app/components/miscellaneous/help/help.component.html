编程语言

  • C 11 [1.0x]

    C 11 [1.0倍运行时间限制]

    • 编译环境：GCC 9
    • 编译标准：GNU C 11
    • 编译参数：-std=c11 -static -march=native -O2 -fno-strict-aliasing -DONLINE_JUDGE
    • 参考程序： 
      #include <stdio.h>
int main() {
  int a, b;
  scanf("%d%d", &a, &b);
  printf("%d\n", a + b);
  return 0;
}

  • Java 11 [2.0x]

    Java 11 [2.0倍运行时间限制]

    • 编译环境：AdoptOpenJDK 11
    • 编译要求：类名必须为Main
    • 编译参数：-J-Xms64m -J-Xmx512m
    • 参考程序： 
      import java.io.*;
import java.util.*;
    
public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        PrintWriter out = new PrintWriter(System.out);
    
        int a = in.nextInt();
        int b = in.nextInt();
        out.println(a + b);
        out.flush();
    }
}

  • Rust [2.5x]

    Rust [2.5倍运行时间限制]

    • 编译环境：Rustc 1.43+
    • 参考程序： 
      use std::io;
    
fn main() {
    let mut input = String::new();
    io::stdin().read_line(&mut input).unwrap();
    let mut s = input.trim().split(' ');
    let a_str = s.next().unwrap();
    let a: i32 = a_str.parse().unwrap();
    let b_str = s.next().unwrap();
    let b: i32 = b_str.parse().unwrap();
    println!("{}", a + b);
}
