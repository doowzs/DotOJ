# 教学使用

## 事前准备

- 安装OJ，ExamId要设置为`null`
- 整个域名，配置HTTPS

## 用户导入

首先拿到全班的名单，然后创建excel表格，填写学号、姓名和邮箱（就用smail好了，反正不会发邮件的）。
将excel表格保存为csv格式，然后导入到OJ批量创建用户。

在上课的时候告诉大家OJ的地址，让大家自行登录修改密码。

## 每周的作业周常

- 每周创建作业，然后出题，出题后创建题目并导入测试数据。
- 题目要注意提前进行自测，有题解的话可以先写好，上传到题解网站。
- 上周作业结束后，在作业成绩页面下载Excel格式的成绩单。
  将成绩中的数据按照学号排序，然后使用替换功能删除所有的`(*)`，即可将得分导入成绩文档。

## 数据错误的情况

如果有同学反映题目数据有误，可以根据他的提交ID查找对应的提交，找到该提交第一个出错的测试点编号，然后再在测试点里面找到对应的测试点文件。
如果数据确实有锅，可以重新出数据，然后在题目管理中重新上传测试数据的zip文件，最后到提交管理中对该题目进行全部重测即可。
